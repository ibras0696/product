# FRONTEND AI — «Паутина истории Чечни»

## 1. Назначение

Документ определяет задачи **Frontend AI** и его субагентов.

Frontend отвечает за:

* интерактивную карту;
* отображение исторических объектов;
* паутину связей;
* поиск;
* фильтры;
* карточки сущностей;
* источники;
* фотографии;
* пользовательские заявки;
* загрузку нескольких фотографий;
* административную модерацию;
* музейный сенсорный режим;
* frontend-тесты.

Перед началом работы Frontend AI и каждый субагент обязаны прочитать и соблюдать существующие локальные правила проекта.

# 3. Frontend AI и субагенты

Координирует frontend-разработку:

* контролирует единый UX;
* проверяет результаты;

# 4. Основные пользовательские сценарии

## Карта

```text
Главный экран
→ карта Чечни
→ выбор населённого пункта
→ карточка
→ открыть паутину
```

## Паутина

```text
Центральный объект
→ связанные люди и места
→ выбор узла
→ раскрытие следующего уровня
→ просмотр источника
```

## Заявка

```text
Выбор типа заявки
→ выбор объекта
→ описание
→ источник
→ фотографии
→ отправка
→ tracking code
```

## Модерация

```text
Очередь
→ заявка
→ проверка данных
→ просмотр фотографий
→ выбор результата
→ approve / reject / revision
```
