# Frontend references

Эта папка хранит только визуальные ориентиры и снятые с production-кода прототипы.
Содержимое `references/` не импортируется из `src/`, не попадает в frontend bundle и не
является источником продуктовых данных.

- `visual/history-web-reference.png` — исходный визуальный ориентир атласа.
- `legacy/landing-showcase/` — прежняя галерея вариантов лендинга, сохранённая для истории.

Единственный источник актуальных UI-токенов находится в `src/shared/ui/tokens.css`.
Runtime-изображения и их лицензии находятся в `public/images/history/`.
